# Tiny To-Do App (Flask + Azure)

This is a simple To-Do List web app built with **Flask**, deployed on **Azure App Service**, and storing data in **Azure Blob Storage**.

## Live App
https://tiny-todo-26021.azurewebsites.net/

## Features
- Add tasks using the add button 
- Toggle completion by clicking a specific task 
- Delete tasks by clicking the "X" button once the task has been created
